# 🔐 Image Steganography Web App

A web application for **hiding and extracting secret messages inside images** using steganography techniques.

🌐 Live Demo:  
https://steganography-vercel.vercel.app/

---

## 📌 Project Overview

This project allows users to hide secret text messages inside images and later extract them securely.  
It uses **LSB (Least Significant Bit) steganography** to embed hidden data in image pixels.

---

## 🚀 Features

- Hide secret messages inside images
- Extract hidden messages from encoded images
- Simple and user-friendly web interface
- Secure message embedding
- Image download after encoding

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Pillow (Python Image Library)

---

## ⚙️ How It Works

1. Upload an image
2. Enter a secret message
3. The system embeds the message inside the image
4. Download the encoded image
5. Use the decode option to extract the hidden message
